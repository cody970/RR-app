COMMON ISSUES FOR HIGH-VALUE WORKS
--------------------------------------------------------------------------------

ISSUE #1: SHARE CONFLICTS
- Symptom: Total writer share > 100%
- Cause: Over-claim by collaborators or publishers
- Resolution:
  1. Identify conflicting shares
  2. Contact collaborators/publishers
  3. Submit correction to BMI
  4. Verify MLC update
  5. Re-submit MLC claim if needed

ISSUE #2: PRO AFFILIATION ERRORS
- Symptom: Writer listed as "NA" or wrong PRO
- Cause: Incorrect PRO data in registration
- Resolution:
  1. Verify writer's actual PRO affiliation
  2. Submit PRO correction to BMI
  3. Verify MLC update
  4. Collaborator must correct at their PRO

ISSUE #3: ISWC MISMATCH
- Symptom: Different ISWCs in BMI vs MLC vs DSPs
- Cause: Multiple ISWC assignments or data entry errors
- Resolution:
  1. Identify correct ISWC from original release
  2. Submit ISWC correction to BMI
  3. Verify MLC update
  4. Contact DSPs to update ISRC-ISWC linkage

ISSUE #4: PENDING SOCIETY REVIEW
- Symptom: Work shows "Pending Society Review" for extended period
- Cause: Conflict between PROs or missing data
- Resolution:
  1. Submit inquiry to BMI
  2. Request expedited review for high-value work
  3. Provide all supporting documentation
  4. Follow up weekly until resolved
