# Contributing to RR-app

Thank you for considering contributing to RR-app! Here are some guidelines to help you get started.

## Developer Guidelines
- Follow the project's coding standards and style guidelines.
- Make sure to keep your changes as small as possible, focusing on one issue at a time.
- Write clear commit messages that explain the purpose of your changes.

## Setup Instructions
1. **Clone the Repository**  
   Use the following command to clone the repo:
   ```bash
   git clone https://github.com/cody970/RR-app.git
   ```
2. **Install Dependencies**  
   Navigate into the project directory and install the necessary dependencies:
   ```bash
   cd RR-app
   npm install
   ```
3. **Run the Application**  
   Start the development server:
   ```bash
   npm start
   ```

## Best Practices for TypeScript
- Use strict typing to avoid unexpected behavior.
- Take advantage of TypeScript interfaces and types to define the expected shape of your data.
- Always declare types for function parameters and return values.

## Testing Guidelines
- Write unit tests for new features to ensure they behave as expected.
- Use Jest or another testing framework as defined in the project.
- Aim for high code coverage and keep tests updated as the codebase evolves.

Thank you for contributing!  
If you have any questions, feel free to reach out to the maintainers.
